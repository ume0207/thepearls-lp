# The Pearls ｜ 集客ランディングページ

神戸三宮の韓国肌管理・フェイシャルワックス専門サロン「**The Pearls（ザ パールズ）**」の
広告用ランディングページ（画像型・教育型ロングLP / スマホ縦長最適化）。

## 特徴

- **画像型ロングLP**：full-bleed のヒーロー／セクション画像の上にテキストを重ねる構成
- **教育型コンテンツ**：「なぜ自己流ケアでは肌が変わらないのか」→「韓国肌管理とは」を解説
- **CV導線**：初回ツヤ肌ワックス体験 **通常¥9,800 → ¥8,800（税込）**、LINE予約・固定CTAドック
- **デザイン**：パール×ブラッシュ×シャンパンゴールドの配色、Shippori Mincho × Cormorant Garamond
- 単一 `index.html`（インラインCSS / Google Fonts CDN）

## 画像

`images/` の5枚は ChatGPT（Codex CLI）の `gpt-image-2` で生成（生成プロンプトは `prompts/`）。

| `hero.png` | ヒーロー（ツヤ肌の女性） |
